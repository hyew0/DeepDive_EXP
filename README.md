## 인터페이스를 사용하여 의존성 주입하기

- Spring 프레임워크에서 인터페이스를 활용하여 의존성을 주입한다. 
- 인터페이스를 정의하고, 구현 클래스를 생성. 
- 인터페이스 기반 의존성 주입 코드를 작성하고, 실행 결과를 확인한다.
